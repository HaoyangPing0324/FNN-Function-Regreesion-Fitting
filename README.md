# 基于 PyTorch 实现全连接神经网络回归拟合

## 任务
使用PyTorch构建并训练一个全连接神经网络，拟合函数

$$
y_1 = \cos(x_1) + \sin(x_2), \\
y_2 = \log(1 + x_1^2 + x_2^2), \\
y_3 = \sqrt{1 + x_1^2 + x_2^2},
$$

其中输入 $x = [x_1, x_2]$ 是二维向量，输出 $y = [y_1, y_2, y_3]$ 是三维向量。在数据生成过程中，为使任务更符合实际情况，需要对每个输出分别添加噪声。


## 要求
### 1、数据生成：
- 在区间$[-6,6]$内随机生成 $N=200$ 个二维输入数据样本 $\boldsymbol{x}$ 。
- 根据上述函数分别计算 $\boldsymbol{y}$ 。
- 使用 `np.random.normal(0,std=***,size=***)` 给 $\boldsymbol{y}$ 随机添加噪声，以模拟真实数据。
- 将最终的数据制作成pytorch数据集，其中70%用于训练，30%用于验证。


### 2、模型设计：
- 构建一个全连接神经网络，包含至少两层全连接层（层数 $L$ 可自行实验测试，每一层的神经元数也自行尝试选择）。
- 输入层输入2个特征，输出层输出3个值。
- 隐藏层中激活函数使用ReLU或Sigmoid（可自行测试）。


### 3、训练过程：
- 损失函数选用均方误差（MSE）。
- 优化器可以选用Adam或SGD，设置合适的学习率。
- 设置合适的batch-size。
- 训练若干个epoch，直至损失收敛，记录每个epoch的训练损失和验证损失。


### 4、结果展示：
- 利用matplotlib绘制训练过程中损失（训练损失和验证损失）随epoch变化的曲线。
- 再次生成少量数据（例如20），比较展示模型预测值与真实（带噪声）目标值的差异（可采用散点图或折线图）。


### 5、代码要求：
- 使用PyTorch完整实现数据生成（包含噪声添加）、模型构建、训练过程以及结果展示。
- 数据生成部分中，可将噪声标准差作为可调参数，便于后续实验和调试。
- 代码中需添加必要的注释，以便理解每个步骤的实现原理和操作流程。
- 电脑如果带有GPU，使用GPU训练。
